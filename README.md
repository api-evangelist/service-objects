# Service ObjectsThis is a repo for managing machine readable artifacts for the [Service Objects APIs](http://www.serviceobjects.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Service Objects APIs](http://www.serviceobjects.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.serviceobjects.com/)- [Developer](http://www.serviceobjects.com/)- [Twitter](https://twitter.com/serviceobjects)- [GitHub](https://github.com/ServiceObjects)The [APIs.json](https://github.com/api-evangelist/service-objects/blob/master/apis.json) index provides links to all the relevant links for the Service Objects APIs, including the OpenAPI and Postman if they are present.
